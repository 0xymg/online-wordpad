A rich text editor lets you format your writing with bold, italic, headings, lists, tables, images, and more — unlike a plain text editor that only supports unformatted characters. When this editor runs in your browser, you get word processor-level formatting without installing any software.

## Plain Text vs Rich Text

Plain text editors like Notepad or nano only handle raw characters. You cannot make a word bold, create a heading, or insert a table. What you type is what you get — nothing more.

A rich text editor gives you formatting control:

| Capability | Plain Text Editor | Rich Text Editor |
|---|---|---|
| Bold, italic, underline | No | Yes |
| Headings | No | Yes |
| Bullet and numbered lists | No | Yes |
| Tables | No | Yes |
| Images | No | Yes |
| Font choices | No | Yes |
| Text color | No | Yes |
| Export to .docx | No | Yes |

If you need to create a document that looks professional — a letter, a report, a resume, meeting notes — you need a rich text editor.
